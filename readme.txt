== Description ==

This plugin adds image-links in a sidebar or in the footer. 
Copy the image resouce in the medialibary, which you like and put in the link in the inputfield.



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy a link from the medialibrary
4. Put the link in the input field in plugin
5. Save your changes
6. Take a widget and put it in the right wigetarea
